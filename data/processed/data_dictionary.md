# Data Dictionary

All processed tables are generated by `scripts/etl_pipeline.py`.

## dim_fund
Source: `01_fund_master.csv`

| Column | Data Type | Business Definition |
|---|---:|---|
| amfi_code | int64 | AMFI scheme identifier used to join fund-level datasets. |
| fund_house | str | Fund house |
| scheme_name | str | Scheme name |
| category | str | Category |
| sub_category | str | Sub category |
| plan | str | Plan |
| launch_date | datetime64[us] | Launch date |
| benchmark | str | Benchmark |
| expense_ratio_pct | float64 | Scheme expense ratio in percent. Validated between 0.1 and 2.5. |
| exit_load_pct | float64 | Exit load pct |
| min_sip_amount | int64 | Min sip amount |
| min_lumpsum_amount | int64 | Min lumpsum amount |
| fund_manager | str | Fund manager |
| risk_category | str | Risk category |
| sebi_category_code | str | Sebi category code |

## fact_nav
Source: `02_nav_history.csv`

| Column | Data Type | Business Definition |
|---|---:|---|
| amfi_code | int64 | AMFI scheme identifier used to join fund-level datasets. |
| date_key | str | YYYY-MM-DD key linking fact tables to dim_date. |
| nav_date | datetime64[us] | Nav date |
| nav | float64 | Net Asset Value of the scheme. Must be greater than zero. |
| daily_return | float64 | Daily percentage change as a decimal, computed from NAV or benchmark close. |
| is_observed_nav | int64 | 1 for original source NAV row, 0 for weekend/holiday forward-fill row. |
